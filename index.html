REST API (FAST API)

This project involved developing REST API endpoints for a fictional business system, allowing efficient data retrieval, updates and deletions for HR, admin and sales operations. It also makes use of Swagger. A handful endpoints were developed such as GET, POST, PUT and DELETE. The API is desgined to handle up to 50 employees records at a time, ensuring performance is optimised. Pagination or rate limiting can be added for larger datasets in the future. This was one of my assignments for my Cloud Computing module.

Custom Image Classification Model (Keras - TensorFlow)

For part of my dissertation, I trained a machine learning model for image classification. I used Keras (TensorFlow) and MobileNetV2 as a base which also used transfer learning from ImageNet. For training I used a customised version of the Oxford102Flowers dataset. I added more images to make the dataset more balanced and added more classes. It was then converted to the CoreML format for efficient use of the Neural Engine found on modern iOS devices.
